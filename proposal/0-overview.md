# Project Proposal

空荧酒馆 - 原神地图 (Kongying Tavern - Genshin Map)

## 1. 项目概览 | Overview

空荧酒馆（Kongying Tavern）致力于为《原神》玩家打造一个比官方地图更极致、更纯粹、更懂玩家的开源地理信息服务工具。我们深知提瓦特大陆的每一寸土地都蕴含着探索的乐趣，因此，我们通过社区共建的方式，将每一个神瞳、每一口宝箱、每一处解谜以最直观、高效的形式呈现。

### 核心承诺：

1. **无商业限制：** 本项目代码完全开源，协议透明，鼓励开发者基于此进行二次开发。

2. **永久免费：** 我们承诺该地图及其衍生工具对玩家永久免费，无任何内购或强制性商业广告。

3. **体验至上：** 我们的目标是超越官方工具的灵活性，提供更多的垂直领域功能。

## 2. 功能特性 | Feature

本项目不仅仅是一个点位标注工具，而是一个完整的地理数据协同系统：

1. **全图点位覆盖：** 包含神瞳、宝箱、采集物、怪物、任务等数千个分类点位，支持层级显示。

2. **多层地图支持：** 针对渊下宫、层岩巨渊、须弥雨林/沙漠等复杂地形，提供精准的多层结构切换。

3. **进度同步与管理：** 支持账号登录，玩家可手动标记已获取点位，数据跨平台同步。

4. **路线规划：** 预设高效采集路线（如狗粮、矿点、突破材料），支持玩家分享自定义路线。

5. **自动追踪：** 采用完全合规的非侵入式图像识别技术，通过分析游戏画面中的小地图特征（OCR 及特征匹配）提取坐标。无需读取游戏内存或封包，即可将玩家在游戏中的实时位置同步反馈至网页端或悬浮窗地图上。支持根据玩家移动轨迹自动绘制探索路线，极大提升点位查漏补缺的效率。

6. **数据可视化：** 通过地图热力图等可视化内容，为玩家揭露提瓦特世界中的那些冷门景点。

## 3. 里程碑目标 | Milestones

我们根据游戏版本迭代与技术路线演进，设定了以下阶段性目标：

| 阶段 | 目标描述 | 关键成果 |
| ---- | ------- | ------- |
| Phase 1 | 基石 | 核心架构搭建与瓦片系统上线，完成提瓦特大地图基础渲染，支持基础点位筛选。
| Phase 2 | 协同 | 用户系统与进度标记功能，实现云端同步，用户可记录已采集点位。
| Phase 3 | 深度 | 上线更加专业化的能力，包括但不限于点位关联、多层地图、自动追踪、可配置交互逻辑等。
| Phase 4 | 开放 | API 开放平台与插件化，允许第三方工具（如悬浮窗、自动识别工具）调用地图数据。
| Phase 5 | 宇宙 | 跨项目迁移与自动化，优化底层框架，使其能快速适配其他同类开放世界游戏。

## 4. 结语

【空荧酒馆-原神地图】不仅是一个工具，更是社区热爱的结晶。我们将持续迭代，确保在提瓦特大陆的每一次冒险中，我们都是您最可靠的向导。
